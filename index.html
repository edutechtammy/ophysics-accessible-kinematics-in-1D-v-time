Velocity vs. Time Graphs — Accessible Interactive Simulation

How to Use This Simulation

This simulation displays a velocity vs. time (v–t) graph for an object moving along a straight line. The graph has three segments defined by four velocity values and three segment durations.

  • Velocity sliders (v₀ – v₃) set the velocity at each transition point, from −20 to 20 m/s.
  • Duration sliders (Δt₁ – Δt₃) set the length of each time segment, from 0.5 to 10 s.
  • When a slider is focused, use ← → arrow keys to adjust by one step, or Page Up / Page Down for larger increments.
  • The Calculated Values table below the graph shows acceleration and displacement for each segment, plus totals.

Blue shading = positive displacement (motion in + direction). Orange shading = negative displacement (motion in − direction).
